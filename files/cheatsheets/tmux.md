# Tmux #

Press CTRL+a and then...


## Tmux Commands ##

* List `ls`
* New `-s <session>`
* Attach att `-t <session>`
* Kill kill-session `-t <session>`
* Rename `rename-session -t <old> <new>`
* Goto `# <0-9>`


## Controls ##

* Choose `w <name>`
* List `=`
* Buffer `PAGEUP/DOWN`
* Command `: <command>`
* Paste `p`
* Copy Mode `ESC`
-Move `hjkl`
-Start `v`
-Copy `y`

## Sessions ##

* New `:new [-s <name>]`
* Rename `$`
* List `s`
* Next `(`
* Previous `)`
* Dettach (leave tmux) `^a d`

## Windows ##

* Create `c`
* List `w`
* Rename `, <name>`
* Last `l`
* Next `n`
* Previous `p`
* Close `&`

## Panes ##

* Split Horizontally `-`
* Split Vertically `|`
* Pane to Window `!`
* Toggle Fullscreen `z`
* Reorganise `SPACE`
* Show nums `q`
* Select `ARROW`
* Previous `{`
* Next `}`
* Switch to other `o`
* Resize `CTRL+ARROW`
* Expand `ALT+ARROW`
* Swap `^o`
* Last `;`
* Kill `x`
